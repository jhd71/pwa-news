Aide

À Propos

  • Portail d'actualités locales regroupant les principaux médias de la région
  • Application installable sur votre appareil pour un accès rapide

Sites & Navigation

  • Cliquez sur une tuile pour ouvrir le site en version mobile
  • Appui long sur une tuile pour accéder aux options (version bureau, modification, suppression)
  • Ajoutez vos propres sites avec le bouton "+" en bas de l'écran

Personnalisation

  • Choisissez entre mode sombre et clair avec le bouton lune/soleil
  • Alternez entre trois vues : grille, grandes tuiles, ou liste
  • Ajustez la taille du texte dans les paramètres selon vos préférences

Chat en Direct

  • Accédez au chat via le bouton flottant en bas à droite
  • Échangez en temps réel avec d'autres utilisateurs
  • Recevez des notifications pour les nouveaux messages (activables dans les paramètres)
  • La conversation est sauvegardée et synchronisée entre vos appareils

Fonctionnalités Pratiques

  • Barre de recherche Google intégrée pour des recherches rapides
  • Menu latéral pour accéder aux liens utiles (météo, sports, etc.)
  • Installation possible sur l'écran d'accueil via le menu latéral
