Installation Simple

Installez l'application en quelques clics sur tous vos appareils

📱

Android

  1. Ouvrez le site dans Chrome
  2. Attendez la bannière d'installation
  3. Ou menu ⋮ → "Installer l'application"
  4. L'app apparaît sur votre écran d'accueil
🍎

iPhone/iPad

  1. Ouvrez dans Safari (obligatoire)
  2. Tapez sur l'icône de partage
  3. Choisissez "Sur l'écran d'accueil"
  4. Confirmez l'ajout
💻

PC/Mac

  1. Icône dans la barre d'adresse
  2. Ou menu → "Installer..."
  3. L'app s'ouvre dans sa fenêtre
  4. Épinglez-la à votre barre des tâches
